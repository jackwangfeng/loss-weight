## 🚀 如何访问

### 方式 1: 本地浏览器访问

直接在浏览器打开：
```

```

### 方式 2: 局域网其他设备访问

1. 获取本机 IP 地址：
```bash
ip addr show | grep "inet " | grep -v 127.0.0.1
```

2. 在其他设备浏览器打开：
```
http://192.168.x.x:8888
```

### 方式 3: 手机访问

确保手机和电脑在同一局域网：
1. 手机浏览器输入：`http://电脑IP:8888`
2. 或使用二维码扫描访问

## ⚠️ 注意事项

### 1. API 连接

当前前端配置的 API 地址是：
```dart
String _baseUrl = 'http://localhost:8000/v1';
```

**问题**：
- 浏览器访问时，`localhost` 指向浏览器所在设备
- 如果后端也在同一台机器运行，可以正常连接
- 如果后端在其他设备，需要修改 API 地址

**解决方案**：
```dart
// 修改为后端服务器的实际 IP 地址
String _baseUrl = 'http://192.168.x.x:8000/v1';
```

### 2. 跨域问题

如果遇到 CORS 错误，确保后端已配置 CORS 中间件。

当前后端已配置 CORS：
```go
r.Use(middleware.CORS())
```

### 3. 端口占用

如果 8888 端口被占用，可以更换其他端口：
```bash
flutter run -d web-server --web-port=9000
```
